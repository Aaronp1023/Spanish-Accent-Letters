# Spanish-Accent-Letters
This script is if you have Spanish classes or just want to use the Spanish accented letters easier. This is a very basic script just using the key binds in autohotkey.

Controls:

Lower Case Hot Keys:
Alt + a = á
Alt + e = é
Alt + i = í
Alt + n = ñ
Alt + o = ó
Alt + u = ú

Upper Case Hot Keys:
Shift + Alt + a = Á
Shift + Alt + e = É
Shift + Alt + i = Í
Shift + Alt + n = Ñ
Shift + Alt + o = Ó
Shift + Alt + u = Ú
 
Special Hot Keys:
Shift + Alt + ? = ¿
Shift + Alt + ! = ¡
